# Robot Puzzle Solver

This project is a small puzzle game where a robot navigates different rooms to solve logic-based challenges.  
The goal is to guide the robot through each room by analyzing the layout, interacting with elements, and finding the correct path to complete the puzzle.

## Controls

- **W** — Move Up
- **A** — Move Left
- **S** — Move Down
- **D** — Move Right
- **E** — Interact (when possible)
- **P** — Pause

## Features

- Puzzle levels
- Robot movement and interaction
- Implemented using persistent and non-persistent managers following the singleton pattern
- New Input System usage
- Basic ilumination and level design
